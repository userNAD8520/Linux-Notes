üìù Quick Reference

  • Symlinks: bin, lib, lib64, sbin point to their /usr equivalents for compatibility
  • Pseudo Filesystems: proc, sys, run are generated by the kernel in-memory
  • User Spaces: /home for regular users, /root for superuser
  • Temporary Storage: /tmp is cleared on reboot, /run contains runtime state
  • System Configuration: /etc contains text-based config files (no binaries)
  • Variable Data: /var for logs, caches, queues that change during operation
